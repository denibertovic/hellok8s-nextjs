# hellok8s-nextjs

Sample nextjs application used for demoing deploying to k8s.[^credits]

What's included:

- Development environment using [devenv](https://devenv.sh) and [nix](https://nixos.org/explore/).
- Dockerfile and Helm chart for deploying to k8s.
- Github actions CI/CD pipeline that builds and deploys the app to a k8s cluster.

[^credits]: Initial scaffolding done with [create-t3-app](https://create.t3.gg/).
